commands : 

-> git config --global user.name "your name"
-> git config --global user.email your email
-> git config --list
-> git init	//to make local repository
-> git add . 	//. to add all the changes into repository
-> git commit -m "commit log here"		//-m means message to track snapshots
-> git log	// shows all the commit history(commit logs)
-> git log -2 	//will only show last 2 commands
-> git status	// if something get change in our project it will show othervise doesn't show anything
-> git clone github url	//copies data to given github url


-> git remote add anyname url of empty repository which you creted in github
-> git push origin master	//push changes to remote server
-> git pull origin master	//pull chages to local system(from github)
-> git pull origin gh-pages

-> git branch
-> git branch branch_name
-> git checkout branch_name	to make it active

-> git push origin brachname
-> git push -u origin textdoc(branch name)
-> git merge branchname (active branch must be master branch)

-> git clone --branch branch_name github repository url
-> git checkout -b feature	//create new branch and make it active
-> git reset filename 
	or just			//to remove it from staging area
-> git reset 		

-> git reset HEAD~1		//last commit will be removed and if we write HEAD~2(HEAD is a pointer)
				//then last two commit will be removed

-> git reset --hard hashcode	//means our latest commit will be this hashcodes commit
				//all latest commit till this hashcodes commit will be gone(erase data)
